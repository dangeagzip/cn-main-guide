# cn-main-guide

## 项目简介

`cn-main-guide` 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定网站、域名或服务，仅作为静态 HTML 页面的集中存放与分发点。

页面内容可能涉及不同主题的说明、指南或工具页面，所有页面均为独立文件，互不依赖。

## 目录说明

仓库结构如下：

```
cn-main-guide/
├── README.md          # 本说明文件
├── index.html         # 仓库首页 / 导航页（可选）
├── pages/             # 独立 HTML 页面目录
│   ├── page-1.html
│   ├── page-2.html
│   └── ...
└── assets/            # 公共资源（CSS、JS、图片等）
    ├── css/
    ├── js/
    └── images/
```

各目录用途：

- `pages/`：存放所有独立 HTML 页面文件，每个文件为一个完整页面。
- `assets/`：存放页面共用的样式、脚本和图片资源，便于统一维护。

## 页面归档说明

本仓库中的 HTML 页面为一次性归档或定期更新内容。每个页面通常：

- 是自包含的 HTML 文件（部分可能引用 `assets/` 下的资源）
- 不依赖外部数据库或后端服务
- 可通过浏览器直接打开或通过 GitHub Pages 等静态托管服务访问

页面内容可能随时间更新或替换。如需查看历史版本，请参考仓库的提交记录。

## 维护说明
